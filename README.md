# Zombie Dice 🧟

An Android dice game built in Java using Android Studio.

## About
Zombie Dice is a press-your-luck dice game where you play as a zombie 
trying to collect 13 brains before the AI opponent. Roll the dice, collect 
brains, but watch out for shotgun blasts!

## How to Play
- Each turn, draw 3 dice from the cup and roll them
- Green dice are easier, yellow are balanced, red are dangerous
- Brain = collect it, Shotgun = danger, Footprint = reroll next turn
- Get 3 shotguns and you lose all brains scored that turn
- Bank your brains to keep them safe
- First to 13 brains wins!

## Features
- Animated dice rolls
- Sound effects
- AI opponent with decision logic
- Victory and defeat screens
- How To Play guide

## Built With
- Java
- Android Studio Panda
- AndroidX ViewModel and LiveData
- Android Animation API
- MediaPlayer for sound
